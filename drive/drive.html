Get Direct Download Links

If you want to process multiple links at once, then add each link on new line.

While pasting any link consider following:

  • For Google-Drive: Right-Click on file => Copy shareable link
  • For Dropbox: Paste 'Share-link'
  • For OneDrive: Paste 'Embed-link'
  • For Github: Paste 'Url-link'

In case of 'Google-Drive' and 'Dropbox' you get two-urls for each link -

  1. Direct-Download URL
  2. File-Hosting URL
